Überblick

Sie geben die API-Version an, die Sie über die URL verwenden möchten, die Ihre Anwendung zum Posten in Zencoder verwendet - Beiträge an https://app.zencoder.com/api/v2/ benutze V2. Wenn keine Version angegeben ist, verwendet die Anforderung die Edge-API, bei der sich die Dinge ohne Vorwarnung ändern können. Verwenden Sie diese Option daher nur, wenn Sie mit einer neuen Funktion experimentieren, bevor sie offiziell verfügbar ist, und niemals in der Produktion.

API-Versionen werden geschlossen, sobald neue Versionen online gehen. Wir geben immer mindestens 6 Monate im Voraus Bescheid, bevor eine Version geschlossen wird, damit Sie Zeit haben, Ihre Anwendung zu migrieren.
