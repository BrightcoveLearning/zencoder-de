Zencoder mit Google Cloud Storage verwenden

In diesem Thema wird das Setup für die Verwendung von Zencoder auf der Google Cloud Platform erläutert.

Google Cloud Storage-Setup

Generieren Sie interoperable GCS-Anmeldeinformationen

Warnung: Die Unterstützung für Google Compute Engine-Regionen wurde am 30. Mai 2017 eingestellt.

Google Cloud Storage ist mit der Amazon S3-API interoperabel, erfordert jedoch einige Ersteinrichtungsarbeiten. Um mit GCS auf Zencoder zu beginnen, müssen Sie aktivieren Interoperabler Zugriff aus der Google Cloud Console und generieren Sie einen Schlüssel. Probier das aus So generieren Sie einen Entwicklerschlüssel Abschnitt in dieses Dokument Anweisungen zum Aktivieren des interoperablen Zugriffs und zum Generieren von Zugriffsschlüsseln.

Anmeldeinformationen speichern

Sobald Sie eine Reihe von interoperablen GCS-Anmeldeinformationen für Ihr Konto haben, können Sie diese sicher bei Zencoder unter speichern Zencoder-Anmeldeinformationen Seite.

Wählen Sie unter Anmeldeinformationen hinzufügen die Option Google Cloud Storage aus. Dann füllen Sie die Google Cloud Storage Access Key ID und Secret Access Key Parameter mit Ihren neu generierten Anmeldeinformationen. Stellen Sie sicher, dass Sie auf "Standardanmeldeinformationen für Übertragungen mit gcs" klicken. Dadurch wird sichergestellt, dass alle Eingabe- oder Ausgabe-URLs, die das angeben gcs:// Das Protokoll verwendet diese Anmeldeinformationen.
