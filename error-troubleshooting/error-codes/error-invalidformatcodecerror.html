Bedeutung

Sie haben einen Codec in einem inkompatiblen Container angegeben. Beispielsweise können Sie den WMV-Codec nicht in einer Datei mit dem Namen "video.mp4" verwenden.

Diesen Job wiederholen?

Jobs mit diesem Fehler sollte nicht erneut versucht werden. Erfahren Sie mehr über das Wiederholen von Jobs.
