Bedeutung

Die für die Eingabedatei berechnete MD5-Prüfsumme stimmt nicht mit der Prüfsumme überein, die der API zur Verfügung gestellt wurde.

Es gibt verschiedene mögliche Ursachen für dieses Problem. Die wahrscheinlichsten sind:

  • Die Datei wurde während der Übertragung beschädigt.
  • Die an die API übermittelte Prüfsumme war falsch.

Problemumgehungen

Stellen Sie sicher, dass die der API übermittelte Prüfsumme mit der Prüfsumme der Datei auf der Festplatte übereinstimmt.

Wenn sie nicht übereinstimmen, führen Sie bitte einen neuen API-Aufruf mit der richtigen Prüfsumme durch.

Wenn die Prüfsummen tun übereinstimmen, wurde die Datei möglicherweise während der Übertragung beschädigt. Bitte reichen Sie den Job erneut ein.

In beiden Fällen kontaktieren Sie uns bitte, wenn der Wiederholungsversuch fehlschlägt.

Diesen Job wiederholen?

Jobs mit diesem Fehler können wiederholt werden. Erfahren Sie mehr über das Wiederholen von Jobs.
