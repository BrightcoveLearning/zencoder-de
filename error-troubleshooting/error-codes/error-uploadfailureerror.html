Bedeutung

Dieser Fehler tritt auf, wenn wir eine Datei aus einem anderen Grund als Zugriff verweigert oder Server nicht gefunden nicht hochladen können. Wenn der Server einen Antwortcode zurückgibt, wird dieser in der Fehlermeldung angezeigt, damit Sie feststellen können, was passiert ist, und die Datei gegebenenfalls aktualisieren können. Kontaktiere uns um mehr Informationen darüber zu erhalten, was passiert ist.

Diesen Job wiederholen?

Jobs mit diesem Fehler sollte nicht erneut versucht werden. Erfahren Sie mehr über das Wiederholen von Jobs.
