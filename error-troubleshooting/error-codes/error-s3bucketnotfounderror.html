Bedeutung

In S3 wurde kein Bucket mit dem angegebenen Namen gefunden. Wenn Sie das s3: // -Protokoll in Ihren URLs verwenden, stellen Sie sicher, dass Sie nur den Bucket-Namen "s3: // my_bucket" und nicht die Domain "s3.amazonaws.com" angeben.

Diesen Job wiederholen?

Jobs mit diesem Fehler sollte nicht erneut versucht werden. Erfahren Sie mehr über das Wiederholen von Jobs.
