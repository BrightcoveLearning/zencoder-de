Bedeutung

Wir haben versucht, eine Datei unter der angegebenen Adresse herunterzuladen, haben jedoch einen Berechtigungsfehler erhalten. Wenn dies eine S3-URL ist, müssen Sie wahrscheinlich Lesezugriff auf Zencoder gewähren für diesen Bucket oder diese Datei. Der einfachste Weg, dies zu tun, ist die Verwendung eines S3-Clients wie der AWS Console von Amazon.

In technischer Hinsicht bedeutet dies, dass wir vom Server eine Antwort mit 403 Zugriff verweigert erhalten haben.

Problemumgehungen

Stellen Sie sicher, dass die Berechtigungen für die Datei unter der angegebenen URL den Zencoder-Zugriff ermöglichen. Wenn sich die Datei in S3 befindet, müssen Sie dies möglicherweise tun Gewähren Sie Lesezugriff auf Zencoder.
