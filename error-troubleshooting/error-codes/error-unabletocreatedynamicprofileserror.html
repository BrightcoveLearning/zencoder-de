Bedeutung

Dieser Fehler tritt auf, wenn bei der Generierung des dynamischen CAE-Profils ein Problem auftritt. Dies bedeutet im Allgemeinen, dass das Profil nicht in einer Weise generiert werden konnte, die den angegebenen Einschränkungen (Wiedergabegrenzen, Bitratengrenzen, Granularitätsgrenzen usw.) entspricht. Wenden Sie sich an den Zencoder-Support Wenn Sie weitere Informationen wünschen.

Diesen Job wiederholen?

Jobs mit diesem Fehler sollte nicht erneut versucht werden. Erfahren Sie mehr über das Wiederholen von Jobs.
