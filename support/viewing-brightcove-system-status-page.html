Brightcove überwacht kontinuierlich den Status aller Brightcove-Dienste und -Anwendungen. Bei Betriebsunterbrechungen wird auf der Systemstatusseite ein Hinweis veröffentlicht. Alle geplanten Wartungshinweise werden ebenfalls auf dieser Seite veröffentlicht.

Um auf die Systemstatusseite zuzugreifen, besuchen Sie https://status.brightcove.com oder klicken Sie in der Navigationskopfzeile auf das Support-Symbol () und dann  Systemstatus.

Alle veröffentlichten Benachrichtigungen enthalten die folgenden Informationen:

  • Vorfallstatus
    • Betriebsbereit (grün)
    • Benachrichtigung (Blue - Systeme sind derzeit in Betrieb, jedoch von Verzögerungen oder zeitweiligen Ausfällen betroffen.)
    • Teilweise Betriebsstörung (gelb)
    • Full-Service-Störung (rot)
  • Komponenten - Service oder betroffener Bereich
  • Standorte
    • APAC
    • Amerika
    • EMEA
    • Japan
  • Aktueller Status und Status. Staaten umfassen:
    • Untersuchung - Ingenieure untersuchen das Problem
    • Identifiziert - Ingenieure haben das Problem identifiziert und versuchen es zu beheben
    • Überwachung - Problem behoben und technische Überwachung, um sicherzustellen, dass alles in Ordnung ist
    • Behoben - Problem behoben

Alle geposteten Nachrichten werden auf Englisch verfasst und die verwendete Zeit ist Eastern Standard Time (GMT-5).

Anzeigen des Systemstatusverlaufs

Klicken Sie auf, um den Systemstatusverlauf anzuzeigen Geschichte Link am Ende der Seite.

Änderungen des Systemstatus abonnieren

Sie können sich anmelden, um benachrichtigt zu werden, wenn sich der Systemstatus ändert. Dieser Abschnitt ist wie folgt organisiert:

  • E-Mail abonnieren
  • Webhook abonnieren
  • Verwalten Sie Ihr Abonnement

E-Mail abonnieren

Um Benachrichtigungen per E-Mail zu erhalten, klicken Sie auf ABONNIEREN Klicken Sie auf die Schaltfläche, geben Sie Ihre E-Mail-Adresse ein und klicken Sie dann auf Abonnieren.

Webhook abonnieren

Ein Webhook ist ein definierter Rückruf mit HTTP POST. Sie geben eine URL an, unter der der Callback ein JSON-Objekt veröffentlicht, wenn eine Statusaktualisierung stattfindet.

Um sich anzumelden, um benachrichtigt zu werden, wenn sich der Systemstatus ändert, wählen Sie die Option ABONNIEREN Taste. Geben Sie dann eine URL ein, um die JSON-Nutzdaten zusammen mit Ihrer E-Mail-Adresse zu erhalten.

Verwalten Sie Ihr Abonnement

Nachdem Sie sich angemeldet haben, erhalten Sie eine E-Mail mit einem Link zu einer Seite "Abonnement verwalten", auf der Sie Ihr Abonnement verwalten können. Sie können die Komponenten / Produkte / Regionen auswählen, für die Sie Benachrichtigungen erhalten möchten. Treffen Sie Ihre Auswahl und klicken Sie dann auf Abonnement speichern.

Um sich abzumelden, klicken Sie auf Abmelden Link unten auf der Seite Abonnement verwalten.

Benachrichtigungen in Slack erhalten

Slack verfügt über ein eigenes Webhook-System, mit dem Statusaktualisierungen verbunden werden können. Konsultieren Sie die Slack-Dokumentation für Details.
