Überblick: Zencoder

Zencoder ist eine leistungsstarke Plattform zum Transcodieren von On-Demand-Videos.

Einführung

Zencoder bietet einen schnellen und zuverlässigen Transcodierungsdienst für Video- und Audiodateien. Es kann Ausgaben für das Web und eine Vielzahl von Geräten erzeugen und kann problemlos in Ihr CMS und andere Anwendungen integriert werden.

Die Benutzeroberfläche

Zencoder ist in erster Linie eine API, aber es gibt eine Benutzeroberfläche, die Folgendes umfasst:

  • Account Einstellungen
  • Ein Dashboard mit den Ergebnissen der letzten Jobs
  • Ein API Request Builder, mit dem Sie mit der API beginnen können, ohne Code schreiben zu müssen

Die API

Es gibt verschiedene Funktionen, die Sie mithilfe der API ausführen können. Die wichtigsten sind das Senden und Verwalten von Codierungsaufträgen für Video-on-Demand.

Eine vollständige Anleitung zu den API-Endpunkten finden Sie in der API-Referenz.

Die Stärke von Zencoder liegt in der Flexibilität, die es Ihnen bietet, genau die Art von Ausgängen zu erstellen, die Sie möchten. Dies ist auch die Quelle einiger Komplexität, da viele Codierungseinstellungen zu verstehen sind. Ein guter Ort, um diese zu erkunden, ist die Zusammenfassung von Codierungseinstellungen Hier finden Sie Links, die Sie zu den vollständigen Beschreibungen mit Beispielcode führen.
