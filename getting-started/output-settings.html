Ausgabeeinstellungen

In diesem Thema werden die Standardausgaben für Zencoder-Jobs und die Grundlagen zum Definieren Ihrer eigenen Ausgaben erläutert.

Ausgänge definieren

Unsere Standardausgabe ist eine einzelne Videodatei mit den folgenden Einstellungen:

H.264 Video, AAC Audio, MP4 Dateiformat
Video: mittlere Qualität
Audio: Stereo, mittlere Qualität (ungefähr 112 kbps), 44100 Hz

Wenn Sie keinen Ausgabespeicherort angeben, wird die Datei vorübergehend von Zencoder gehostet. Nach 24 Stunden ist die Datei nicht mehr verfügbar.

Natürlich können Sie das steuern Ausgabeeinstellungen. Übergeben Sie dazu eine Reihe von Optionen in der API-Anforderung wie folgt:

{
  "api_key": "93h630j1dsyshjef620qlkavnmzui3",
  "input": "s3://bucket-name/file-name.avi",
  "outputs": [
    {
      "url": "s3://output-bucket/output-file-name.mp4",
      "width": "1280",
      "height": "720"
    }
  ]
}

In diesem Beispiel wird eine einzelne Ausgabedatei aus dem ursprünglichen Eingabevideo erstellt und in einen S3-Bucket hochgeladen. Sie können erstellen mehrere Ausgabedateien vom Original, ohne zusätzliche Codierungsjobs erstellen zu müssen.
