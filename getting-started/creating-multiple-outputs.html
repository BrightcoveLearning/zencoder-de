Mehrere Ausgaben erstellen

In diesem Thema wird erläutert, wie Sie mehrere Ausgaben für einen Codierungsjob erstellen.

Erstellen Sie mehrere Ausgänge

Sie können mehrere Ausgaben angeben, indem Sie mehr als einen Hash im Ausgabearray übergeben, wie folgt:

{
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}
