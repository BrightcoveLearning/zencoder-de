Hinweis

hint:Boolean

API-Versionen: V1, V2

Elternteil: Ausgabe

Standard: falsch

Gültige Werte: richtig oder falsch

Kompatible Auftragstypen: Alle

Beispiel: wahr

Beschreibung:

Aktivieren Sie den Hinweis auf MP4-Dateien für RTP / RTSP. Der Standard mtu_size ist 1450 Bytes; benutze die mtu_size Option, dies zu ändern.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

mtu_size

mtu_size:Integer

API-Versionen: V1, V2

Elternteil: Ausgabe

Standard: 1450

Gültige Werte: Eine ganze Zahl zwischen 100 und 5000

Kompatible Auftragstypen: Alle

Beispiel: 2000

Beschreibung:

MTU-Größe in Byte für MP4-Hinweise.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}
