Kodierungsempfehlungen

AV1 ist ein offenes, lizenzgebührenfreies Videokodierungsformat der nächsten Generation der Alliance of Open Media Video. Es kann eine überlegene Qualität bei deutlich niedrigeren Bitraten im Vergleich zu HEVC-, VP9- und H.264-Codecs liefern.

Siehe die AV1-Handbuch Für weitere Informationen und achten Sie besonders auf die Einschränkungen für aktuelle Unterstützung.

Zusätzlich zu den Einstellungen auf dieser Seite möchten Sie möglicherweise andere allgemeine Videoeinstellungen hinzufügen. Beachten Sie jedoch, dass die folgenden Einstellungen gelten nicht Derzeit unterstützt:

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

Beachten Sie auch, dass für Auflösungseinstellungen (ob Sie size oder width und height) Sowohl die Breite als auch die Höhe müssen ein Vielfaches von 8 sein.

video_codec_level

video_codec_level:String

API-Versionen: V2

Elternteil: Ausgänge

Gültige Werte:

  • 2.0
  • 2.1
  • 3,0
  • 3.1
  • 4,0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

Kompatible Auftragstypen: Alle

Beispiel: 4.1

Beschreibung:

Beschränkt die Bitrate und die Coding Tree Units (CTUs) (AV1).

Beachten Sie, dass diese Einstellung für andere Codecs mit anderen zulässigen Werten verwendet werden kann. Sehen Einstellungen für die Videokodierung.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

video_codec_profile

video_codec_profile:String

API-Versionen: V2

Elternteil: Ausgänge

Gültige Werte: main, high, professional

Standard: main

Kompatible Auftragstypen: Alle

Beispiel: 4.1

Beschreibung:

Legt das Codierungsprofil fest.

Beachten Sie, dass diese Einstellung für andere Codecs mit anderen zulässigen Werten verwendet werden kann. Sehen Einstellungen für die Videokodierung.

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}
