start_clip

start_clip:String

API-Versionen: V1, V2

Elternteil: Ausgänge

Gültige Werte: Ein Timecode (HH: MM: SSS) oder eine positive Zahl, die eine Anzahl von Sekunden darstellt

Kompatible Auftragstypen: vod

Beispiel:

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12,5

Beschreibung:

Erstellen Sie einen Unterclip des Originalvideos, beginnend zu einem bestimmten Zeitpunkt. Dies kann entweder ein Timecode mit dem Format HH: MM: SSS oder eine Dezimalzahl sein, die die Anzahl der Sekunden ausdrückt, bei der begonnen werden soll. Beachten Sie, dass die tatsächliche Startzeit je nach Anordnung der Keyframes im Video möglicherweise nicht genau ist.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "start_clip": "00:00:10.0" }] }

clip_length

clip_length:String

API-Versionen: V1, V2

Elternteil: Ausgänge

Gültige Werte: Eine Dauer im Format HH:MM:SSS oder eine positive Zahl, die eine Anzahl von Sekunden darstellt

Kompatible Auftragstypen: Alle

Beispiel:

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12,5

Beschreibung:

Erstellen Sie einen Unterclip mit einer bestimmten Länge. Dies kann entweder eine Dauer mit einem Format von HH: MM: SSS oder eine Dezimalzahl sein, die die Länge des Clips in Sekunden ausdrückt. Kann mit kombiniert werden start_clip um Unterclips in der Mitte eines Videos zu erstellen. Wenn start_clip Wird dies nicht bereitgestellt, beginnt der Clip am Anfang des Videos. Beachten Sie, dass die tatsächliche Länge des Clips je nach Anordnung der Keyframes im Video möglicherweise nicht genau ist. Beachten Sie auch, dass clip_length eine Dauer ist und nicht der Timecode in der Mediendatei, an dem der Subclip enden soll.

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "clip_length": "00:00:10.0" }] }
