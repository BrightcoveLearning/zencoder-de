AVC-Intra Übersicht

AVC-Intra definiert zwei Voreinstellungen, AVC-Intra 50 und AVC-Intra 100. AVC-Intra 50 wird mit einer Bitrate von 50 MBit / s codiert, während AVC-Intra 100 mit 100 MBit / s codiert wird. Verwenden Sie die video_codec_preset Parameter zum Definieren eines AVCi-Ausgangs. Informationen zum Erstellen von AS-11-Ausgängen finden Sie in der Dokumentation Hier.

  • video_codec_preset:: Legt die AVC-Intra-Voreinstellung für die Ausgänge fest. Werte: avci_50, avci_100.

Beispieleinstellungen

Beispiel AVC-Intra 50-Anfrage

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_50"
    }
}

Beispiel AVC-Intra 100-Anfrage

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_100"
    }
}
