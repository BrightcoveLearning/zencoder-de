    Leitfaden für Audio-Mapping

    In diesem Thema werden Optionen für die Audiomappung für mehrere Audiospuren erläutert.

    Einführung

    Es ist üblich, dass Bearbeitungssysteme Videodateien ausgeben, die mehrere Audiospuren enthalten - entweder für Surround-Sound-Audio-, Archiv- oder mehrsprachige Szenarien. Mit Zencoder können Eingangsdateien, die mehrere Audiospuren enthalten, auf eine einzige Audiospur am Ausgang abgebildet oder gemischt werden. Diese Optionen können beispielsweise kombiniert werden, um 6 diskrete Mono-PCM-Audiospuren von einer Eingabedatei in eine einzige 5.1-Surround-Sound-Audiospur an einem Ausgang abzubilden.

    In diesem Handbuch werden zwei Optionen vorgestellt: input_audio_channels und mit denen gesteuert werden kann output_audio_channels , wie Audiospuren und Kanäle aus einer Eingabedatei in die Ausgabedatei abgebildet werden.

    Unterstützte Eingangs-Audiok

    Die folgenden Audiokanäle werden für den Import unterstützt:

    • FL
    • FR
    • FC
    • LFE
    • BL
    • BR
    • Lt (FL-Matrix)
    • Rt (FR-Matrix)

    Erstellen Sie mehrere Ausgaben von einer Quelle mit mehreren Audiospuren

    Ein ziemlich häufiger Anwendungsfall ist ein Quellvideo mit mehreren Audiospuren, und Sie möchten mehrere Audiodateien mit jeweils einer der Audiospuren ausgeben. Hier ist der beste Weg, um die Ausgaben dafür zu konfigurieren:

    outputs: [
  {
  label: "Output 1",
  url: "your hosting server for outputs/output1.mp4",
  skip_video: true,
  "input_audio_service_types":{"1": "main"},
  "select_audio_service_type": "main"
  },
  {
  label: "Output 2",
  url: "your hosting server for outputs/output2.mp4",
  skip_video: true,
  "input_audio_service_types":{"2": "main"},
  "select_audio_service_type": "main"
  },
  {
  label: "Output 3",
  url: "your hosting server for outputs/output3.mp4",
  skip_video: true,
  "input_audio_service_types":{"3": "main"},
  "select_audio_service_type": "main"
  } ...

    Optionen für die Audio-Zuordnung

    • input_audio_channels: Legen Sie das Layout der Audiospuren zu Kanälen für den Eingang fest.
    • output_audio_channels: Stellen Sie das Layout des Ausgangskanals ein.

    Die input_audio_channels Option kann verwendet werden, um einzelne Audiospuren oder Kanäle aus der Eingabedatei zu markieren, indem ihnen ein Kanalname gegeben wird, wie FL für den Kanal „vorne links“ in einer 5.1-Surround-Audiospur. Die folgenden Beispiele zeigen, wie Sie diese Option verwenden, um von 6 einzelnen Audiospuren zu einer einzelnen 5.1-Surround-Spur abzubilden.

    Durch Festlegen der output_audio_channels Option kann das Kanallayout der Ausgabe-Audiospur angepasst werden. Wenn dies nicht festgelegt ist, wird die Standardanordnung für den Audiocodec verwendet.

    Die output_audio_channels Option kann auf zwei Arten verwendet werden: um einen Mixdown aller Kanäle in zu erstellen oder um eine Ausgabe mit Audiokanälen zu erstellen input_audio_channels , die einem benutzerdefinierten Ausgabekanal-Layout zugeordnet ist, ohne dass ein Mixing durchgeführt wird. Zencoder kann einen Stereo- oder Mono-Mixdown aus einer Audiospur mit Mehrkanal- oder Surround-Sound-Eingang erzeugen. Um einen Mixdown-Ausgang zu erstellen, setzen Sie diese Option entweder auf 1 (für Mono) oder 2 (für Stereo). Um ein benutzerdefiniertes Kanallayout zu erstellen, legen Sie diese Option auf ein Array von Track-IDs fest, die in der input_audio_channels Option definiert wurden. In den folgenden Beispielen finden Sie weitere Informationen.

    Zencoder unterstützt derzeit Mono-, Stereo- und 5.1-Kanal-Ausgabe-Audio.

    Stereo-Mixdowns können aus Surround-Audio mit beiden Tasten erzeugt werden input_audio_channels und output_audio_channels Optionen. Durch output_audio_channels die Einstellung auf 2 ist es möglich, einen Stereo-Mixdown zu erstellen.

    Beispiel für Einstellungen

    5.1 Surround Track von mehreren Mono-Tracks

    In diesem Beispiel werden einzelne Audiospuren den benannten Kanälen zugeordnet FL, FR, FC, LFE, BL, BR und eine einzige 5.1-Surround-Soundspur aus dem Mapping erstellt.

    {
  "input": "s3://zencodertesting/test-multitrack-audio.mov",
  "output": {
    "input_audio_channels": {
      "FL": 1,
      "FR": 2,
      "FC": 3,
      "LFE": 4,
      "BL": 5,
      "BR": 6
    }
  }
}

    Stereo-Mixdown von einem Mapped 5.1-Surround-Track

    Das folgende Beispiel zeigt, wie Sie einen Stereo-Mixdown aus der 5.1-Surround-Audiospur erstellen, die im obigen Beispiel erstellt wurde.

    {
  "input": "s3://zencodertesting/test-multitrack-audio.mov",
  "output": {
    "input_audio_channels": {
      "FL": 1,
      "FR": 2,
      "FC": 3,
      "LFE": 4,
      "BL": 5,
      "BR": 6
    },
    "output_audio_channels": 2
  }
}

    Benutzerdefiniertes Audiokanal-L

    Die output_audio_channels Option kann verwendet werden, um zusätzlich zum Erstellen von Mixdowns ein benutzerdefiniertes Kanallayout zu erstellen. Im folgenden Beispiel wird ein Stereoausgang aus dem Front-Links- und Front-Rechts-Kanäle erstellt, wobei jedoch die Reihenfolge vertauscht wird, sodass der linke Kanal rechts ist und umgekehrt.

    {
  "input": "s3://zencodertesting/test-multitrack-audio.mov",
  "output": {
    "input_audio_channels": {
      "FL": 1,
      "FR": 2,
      "FC": 3,
      "LFE": 4,
      "BL": 5,
      "BR": 6
    },
    "output_audio_channels": ["FR", "FL"]
  }
}

    Unterstützung mehrerer Sprachspuren

    Durch die Verwendung der input_audio_channels Option für eine Eingabe, die mehrere Sprachspuren enthält, ist es möglich, mehrere Ausgaben mit jeweils separaten Sprachspuren zu erstellen. Angenommen, eine Eingabedatei hatte 4 diskrete Audiospuren und dieser Titel 1 enthielt das englische (linke) Audio, Track 2 enthielt das englische (rechte) Audio, Track 3 enthielt das spanische (linke) Audio und Track 4 enthielt das spanische (rechte) Audio. Indem Sie den input_audio_channels Parameter für jede Ausgabe unterschiedlich definieren, ist es möglich, mehrere Sprachausgaben aus einer einzelnen Datei zu erstellen.

    {
  "input": "s3://zencodertesting/test-multitrack-audio.mov",
  "outputs": [
    {
      "label": "english",
      "input_audio_channels": {
        "FL": 1,
        "FR": 2
      }
    },
    {
      "label": "spanish",
      "input_audio_channels": {
        "FL": 3,
        "FR": 4
      }
    }
  ]
}

    Benutzerdefiniertes Kanal-Mapping

    Das folgende Beispiel zeigt, wie ein vorhandener Stereo-Downmix aus einer Mediendatei extrahiert wird, in der die erste Audiospur sowohl einen 5.1-Surround-Mix als auch einen Stereo-Downmix (wo Stereopaar) enthält.

    {
  "input": "s3://zencodertesting/test-multichannel-audio.mov",
  "output": {
    "input_audio_channels": {
      "FL": "1:7",
      "FR": "1:8"
    },
    "output_audio_channels": 2
  }
}
