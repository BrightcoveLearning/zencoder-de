Einführung

Bei der Codierung von Videoausgängen für das Streaming wird die Verwendung nicht empfohlen Qualität Einstellung allein (dies verwendet den CRF-Codierungsmodus). Wenn Sie den qualitätsorientierten Modus bevorzugen, empfehlen wir dringend, die VBV-Einstellungen zu verwenden, um die Variation der Bitrate so zu beschränken, dass sie den Spezifikationen des Streaming-Formats entspricht.

VBV-Einstellungen

  • decoder_bitrate_cap
  • decoder_buffer_size

Andere empfohlene Einstellungen zu berücksichtigen

  • video_reference_frames
  • video_bframes

Beispiel

Hier ist ein Beispiel für eine qualitätsorientierte Codierung mit VBV-Einschränkungen.

{
    "type":"segmented",
    "width":640,
    "height":360,
    "quality":3,
    "video_codec_profile":"main",
    "video_reference_frames": 4,
    "video_bframes": 3,
    "decoder_bitrate_cap": 1440,
    "decoder_buffer_size": 1920,
    "keyframe_rate": 0.5,
    "max_frame_rate": 30,
    "audio_quality":5,
    "segment_seconds":10
}
