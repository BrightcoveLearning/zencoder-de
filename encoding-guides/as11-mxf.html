Überblick

Um einen AS-11-Ausgang zu erstellen, stellen Sie die ein format_preset Parameter auf "as11_hd". Derzeit ist Audio auf 2-Kanal-PCM-Audio mit einer maximalen Abtastrate von 48 kHz beschränkt. Weitere Informationen zu AVC-Intra-Ausgängen finden Sie in der Dokumentation Hier.

  • format_preset:: Stellen Sie dies für einen AVCi 100 MXF-Ausgang auf as11_hd ein.
  • audio_sample_rate:: Die Standardeinstellung ist nicht festgelegt. Limitiert auf 48000.

Beispieleinstellungen

Beispiel AS-11-Anfrage

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "format_preset": "as11_hd",
        "audio_sample_rate": 48000
    }
}
